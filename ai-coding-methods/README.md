## 当前最佳实践 (2025-03)

### 推荐工作流

```
1. Kimi / Perplexity → 需求分析和技术调研
2. Cursor / Windsurf → 代码生成和调试
3. 手动优化 → 精细调整
4. 测试验证 → 部署
```

### 核心原则

| 原则 | 说明 |
|------|------|
| **AI 辅助而非替代** | AI 生成 + 人类判断 |
| **渐进式编程** | 小步迭代，快速验证 |
| **上下文积累** | 使用 Context Hub 记录学习 |
| **多工具组合** | 根据场景选择最佳工具 |
